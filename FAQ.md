# ClassPush Pro 常见问题排查指南 (Troubleshooting)

为了帮助大家更顺畅地使用软件，这里汇总了同学们在安装和使用过程中可能遇到的问题及解决方案。

---

## 1. 安装拦截问题

### 🔴 现象：安装时提示 "Unable to execute file in the temporary directory..."
**报错截图**：
> Error 4551: 应用程序控制策略已阻止此文件

**原因**：这是因为你的电脑系统（通常是公司电脑或经过“优化”的系统）开启了严格的安全策略，禁止从临时文件夹运行安装程序。

**✅ 解决方案（按顺序尝试）：**
1.  **右键 -> 以管理员身份运行**（90% 的人能解决）。
2.  **暂时关闭杀毒软件**（如 360、火绒），安装完再打开。
3.  **换个位置**：不要在微信/QQ 文件传输助手里直接点开。把安装包**复制到 D 盘根目录**，然后再右键管理员运行。

---

### ⚠️ 现象：浏览器下载提示 "通常不会下载...请确保信任"
**报错截图**：
> Edge/Chrome 提示：通常不会下载 ClassPush_Setup.exe，请在打开前确保信任...

**原因**：这**不是病毒**！是因为 ClassPush 是一个全新的小众软件，微软的数据库里还没有收录它的“信誉分”，所以浏览器会出于谨慎提示风险。

**✅ 解决方案：**
1.  鼠标移到提示条上，点击右侧的 **... (更多)**。
2.  选择 **保留 (Keep)**。
3.  在弹出的确认框中，点击 **显示详细信息** -> **仍要保留**。
4.  下载完成后即可正常安装。

## 2. 运行白屏问题

### ⚪ 现象：打开软件后窗口是白色的，什么都不显示
**原因**：ClassPush 依赖 Windows 系统的 **WebView2 (Edge)** 内核来显示界面。

**情况一：Win7 / 旧版 Win10 (系统确实缺组件)**
*   **表现**：打开就白屏，或者提示缺少 DLL。
*   **✅ 解决方案**：
    1.  **下载微软官方补丁**：[点击下载 WebView2 运行库](https://go.microsoft.com/fwlink/p/?LinkId=2124703)
    2.  安装下载好的 `MicrosoftEdgeWebview2Setup.exe`。
    3.  重新打开 ClassPush 即可。

**情况二：Win11 / 新版 Win10 (明明有组件，还是白屏)**
*   **表现**：安装补丁时提示“已安装更高版本”，但软件打开还是白屏。
*   **原因**：通常是因为软件放在了**受保护的目录**（如 D 盘根目录），或者是**旧的配置文件冲突**。
*   **✅ 解决方案**：
    1.  **换个目录**：不要直接放在 D 盘根目录。**新建一个文件夹**（如 `D:\Soft\ClassPush`）把软件放进去，或者直接放在**桌面**上运行。
    2.  **清理配置**：打开文件资源管理器，进入 `C:\Users\你的用户名\.ClassPush`，把里面的 `config.json` 删掉，然后重新打开软件。
    3.  **兼容模式**：右键 `ClassPush.exe` -> 属性 -> 兼容性 -> 勾选“以兼容模式运行(Win8)”，并**取消勾选**“以管理员身份运行”。

## 3. 推送问题

### 🔔 现象：设置了自动推送，但一直没收到消息
**原因**：通常是因为配置没填全，或者没点保存。
*   注意：必须填写 **WxPusher UID** (不是微信 ID，也不是微信号)。
*   注意：必须填写 **学号和密码**。

**✅ 解决方案：**
1.  打开软件，进入“设置”页面。
2.  **重新输入**学号、密码、UID（确保没填错）。
3.  **关键动作**：点击页面下方的 **“保存并登录”** 按钮。
4.  保存成功后，点击 **“立即推送”** 测试一下。如果手机收到了，说明配置成功。

## 4. 其他疑问

**Q: 360 提示有病毒？**
A: 这是**误报**。因为软件是用 Python 打包的 (`PyInstaller`)，且没有购买昂贵的数字签名证书，容易被国产杀毒软件误判。
*   请添加信任，或在运行时暂时关闭杀毒软件。
*   如果不放心，可以去 [GitHub 查看源代码](https://github.com/King52HerTz/ClassPushPro) 自己编译。
